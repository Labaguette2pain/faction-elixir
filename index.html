Modèle d'Acteurs

Chaque opération tourne dans un processus BEAM isolé et ultra-léger — des millions en parallèle sans fil d'exécution partagé. La défaillance d'un acteur ne cascade jamais : les superviseurs OTP relancent les processus en microsecondes, automatiquement.

# Spawn : 1M processus en ~1s sur BEAM

Phoenix LiveView

Le framework web temps réel de la faction. Phoenix (LiveView) maintient des connexions WebSocket persistantes sans JavaScript côté client — les mises à jour DOM arrivent du serveur en millisecondes. Zéro round-trip API, zéro état client à synchroniser.

~H sigil → HTML natif, réactivité serveur

Immuabilité Structurelle

Elixir élimine la catégorie entière des bugs de concurrence par design : les données sont immuables. Aucun état partagé mutable, aucun mutex à gérer, aucune condition de course possible. Le partage de données entre processus se fait par message — toujours sûr, toujours prévisible.

state = Map.put(state, :key, val) # nouveau state
02 // RIVALITÉS

JavaScript

L'usine à gaz du Web

MENACE

Écosystème brisé

node_modules peut dépasser le poids de l'application entière. Des centaines de micro-paquets jamais maintenus créent une surface d'attaque et une dette technique permanentes.

Typage chaotique

La coercition de types dynamique produit des aberrations comme [] == ![] évalué à true. TypeScript plâtre le problème sans l'éliminer à la racine.

Instabilité sous charge

Le modèle mono-threadé de Node.js avec event loop s'effondre face à des charges I/O massives simultanées. La concurrence réelle nécessite des workers complexes à orchestrer manuellement.

Rust

La forteresse inflexible

MENACE

Le Borrow Checker

La guerre permanente avec le Borrow Checker transforme le prototypage en cauchemar. Les concepts d'ownership, lifetime et borrow nécessitent des semaines d'apprentissage avant d'être productif sur du code non-trivial.

Compilation interminable

Les temps de compilation explosent sur les projets de taille réelle. Le cycle build-test ralentit drastiquement l'itération. cargo check atténue partiellement, mais la pénalité reste structurelle.

Zéro flexibilité

Rust est taillé pour les systèmes critiques bas niveau, pas pour l'itération rapide. Le coût cognitif permanent de la gestion mémoire manuelle annule le bénéfice pour tout projet web ou business logic standard.
